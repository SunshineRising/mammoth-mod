# Mammoth Mod for Luanti

Adds a woolly mammoth mob to Luanti using the [Mobs Redo](https://content.luanti.org/packages/TenPlus1/mobs/) framework.

## Features

- **Neutral animal** — peaceful until attacked, then the whole herd fights back
- **Snowy biome spawns** — found on snow, ice, and frozen dirt
- **Tameable** — feed wheat or dry grass 10 times to tame
- **Breedable** — two tamed mammoths can produce a baby
- **Capturable** — lasso (60%) or net (5%)
- **Drops** — raw meat, leather, and ivory tusks (50% chance)
- **3D model** — animated elephant mesh with woolly brown texture and white tusks

## Installation

Clone directly into your Luanti mods directory:

```
cd ~/.minetest/mods
git clone https://github.com/SunshineRising/mammoth-mod.git mammoth
```

**Requires:** [Mobs Redo](https://content.luanti.org/packages/TenPlus1/mobs/) and the default game.
